# 🌤 Weather Bot — Погодный Telegram-бот

Telegram-бот для получения и обработки погодной информации из интернета.  
Разработан в рамках семестровой работы по курсу **«Введение в интеллектуальные системы»**.

---

## 📌 Описание проекта

Погодный чат-бот в Telegram, который позволяет пользователю получать актуальную информацию о погоде по любому городу мира, просматривать прогноз на 5 дней, читать свежие новости о погоде, а также просматривать историю своих запросов.

---

## ⚙️ Реализованные функции

| № | Функция | Механизм получения данных |
|---|---------|--------------------------|
| 1 | 🌤 Текущая погода по городу | HTTP API (OpenWeatherMap) |
| 2 | 📅 Прогноз погоды на 5 дней | HTTP API (OpenWeatherMap) |
| 3 | 📰 Новости о погоде | Скрапинг HTML (gismeteo.ru) |
| 4 | 📋 История запросов пользователя | Чтение лог-файла `{ID}.log` |

---

## 🛠 Используемые технологии

- **Python 3.14**
- **python-telegram-bot** — библиотека для работы с Telegram Bot API
- **requests** — HTTP-запросы к внешним сервисам
- **BeautifulSoup4** — парсинг HTML-страниц
- **OpenWeatherMap API** — источник погодных данных

## 🚀 Установка и запуск

### 1. Клонируй репозиторий
```bash
git clone https://github.com/ArtemZaslavskii/weather-bot.git
cd weather-bot
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Создай файл `config.py`
```python
TELEGRAM_TOKEN = "твой_токен_от_BotFather"
OWM_API_KEY = "твой_ключ_от_OpenWeatherMap"
```

### 4. Запусти бота
```bash
python main.py
```

### 5. Открой Telegram
Найди своего бота по username и напиши `/start`

## 📝 Система логирования

Для каждого пользователя автоматически создаётся файл `logs/{ID}.log`, где `ID` — уникальный идентификатор пользователя в Telegram. В файле хранится полная история взаимодействия: все запросы, действия и результаты с временными метками.

Пример содержимого лог-файла:
```
[2026-06-03 14:22:01] START: @username
[2026-06-03 14:22:15] WEATHER_REQUEST: Москва
[2026-06-03 14:22:15] WEATHER_RESULT: 🌤 Погода в Moscow, RU: 🌡 Температура: 18°C
[2026-06-03 14:23:04] FORECAST_REQUEST: Париж
[2026-06-03 14:23:05] FORECAST_RESULT: 📅 Прогноз на 5 дней для Paris, FR
```

## 🔑 Получение ключей

**Telegram Bot Token:**
1. Найди в Telegram @BotFather
2. Напиши `/newbot` и следуй инструкциям
3. Скопируй полученный токен

**OpenWeatherMap API Key:**
1. Зарегистрируйся на [openweathermap.org](https://openweathermap.org)
2. Перейди в раздел API keys
3. Скопируй ключ (активируется в течение 10 минут)
